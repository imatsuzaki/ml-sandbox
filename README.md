## Prerequisites

 - docker
 - docker-compose
 
## Setup

```shell script
$ docker-compose build --no-cache
$ docker-compose up -d
```

Connect docker container from [PyCharm](https://www.jetbrains.com/help/pycharm/using-docker-as-a-remote-interpreter.html#) or vscode.

## Stop

```shell script
$ docker-compose stop
```
